呼吸心球 (Breath Ball) ── 情绪疗愈微游戏
一款专为情绪焦虑、烦躁或需要心理疗愈的人群设计的手机端网页小游戏。通过结合心理学呼吸机制（4-7-8呼吸法变体）与物理弹射解压反馈，帮助用户在指尖的按压、拖拽与丝滑的碰撞中舒缓压力，找回内心的平静。

🧘‍♂️ 心理学设计核心：呼吸规律
本游戏基于临床心理学推荐的“正念呼吸法”设计。通过规律的视觉指引，引导用户调节副交感神经系统，降低心率：

吸气阶段（建议 4 秒）：长按屏幕。视觉上小球如肺部般缓缓胀大，引导深吸气，注入能量。

呼气/弹射阶段（建议 6 秒）：松开手指。小球化为能量体弹射而出，在屏幕间丝滑碰撞反射。此阶段引导用户全身心放松，将负面情绪随小球一起“发射”并释放。

🕹️ 游戏核心玩法与特性
1. 动态呼吸精确计数器 
初始状态：刚进入游戏时处于静止默认阶段，计数器显示为 0 / 4s 吸气，此时不进行秒数计时，不给用户带来任何数字焦虑。

吸气计时：当手指第一次按下屏幕时，秒数动态启动。屏幕实时显示已吸气秒数（严格以整数显示：1秒、2秒，无繁杂的小数点）。

呼气计时：当手指松开后，立刻转为呼气阶段，建议时间为 6秒。计数器动态更新已呼气时间。

归位重启：当呼气满 6 秒（或弹射能量释放完毕）后，游戏自动恢复到不计数的默认初始阶段，静静等待用户的下一次触碰。

2. 蓄力弹射与拖拽控制 (趣味性增强)
为了摆脱单纯呼吸的枯燥感，游戏融入了基于物理引擎的趣味互动：

蓄力机制（按得越久，飞得越快）：吸气（长按）的时间越长、力气越大，小球内部积蓄的“核心能量”就越强。松手刹那的初始弹射速度会呈现正比爆发，随后在丝滑的减速中治愈视觉。

拖拽定向（往哪拖，往哪飞）：打破原有的随机弹射。在从“吸气”到“呼气”松手的那一瞬间，系统会精准捕捉你手指最后拖拽滑动的方向，小球会顺着你手指指引的方向精准飞出，实现真正的“指哪打哪”。

物理碰撞反射：小球在屏幕内移动时，触碰到手机边缘会自动根据动量守恒和物理反射角完美反弹，像解压沙漏一样在屏幕内优雅律动。

3. 视觉动效：色彩柔和，UI 极简，去除一切干扰视觉的乱象，小球运动轨迹平滑不刺眼，拒绝眼花缭乱。

无外部依赖：单文件/轻量化结构，支持任何手机、电脑浏览器直接打开。
